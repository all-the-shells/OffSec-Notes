# Basic info collection
## Host enumeration
```
cat /etc/issue 
cat /proc/version 
hostname 
uname -a
```
## Users
```
cat /etc/passwd
cat /etc/shadow
id
who
w
sudo -l
su -l
useraccount ALL=/usr/bin/apt-get, /sbin/shutdown
```
## Apps & services
```
ps aux
ps aux | grep root
dpkg -l
rpm -qa
ls -ls /etc/ | grep .conf
ls -ls /var/www/html/
find /* -user root -perm -4000 -print 2>/dev/null
```
